# Proyecto - Tienda Online

## Descripción

Aplicación web desarrollada con HTML, CSS y JavaScript que permite visualizar productos, agregarlos al carrito de compras y gestionar una compra.

## Tecnologías utilizadas

* HTML
* CSS
* JavaScript
* Git
* GitHub
* Docker

## Requisitos previos

* Docker Desktop
* Git

## Instalación

Clonar el repositorio:

git clone https://github.com/marinellijoana1-source/Proyecto-.git

## Construcción de la imagen Docker

docker build -t proyecto-web .

## Ejecución del contenedor

docker run -d -p 8080:80 --name proyecto-web-container proyecto-web

## Acceso a la aplicación

Abrir en el navegador:

http://localhost:8080
